# Quiz-Game

# 🎯 JavaScript Quiz Game

A simple and interactive Quiz Game built using HTML, CSS, and JavaScript. This project allows users to test their knowledge through multiple-choice questions and receive their final score instantly.

# 🚀 Features
Multiple-choice quiz questions
Real-time score calculation
Next question navigation
Responsive and user-friendly interface
Instant result display
Easy to customize and add new questions
# 🛠️ Technologies Used
 - HTML5
 - CSS3
 - JavaScript (ES6)
